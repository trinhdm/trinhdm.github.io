Foreground Studios, a subsidiary of Foreground Company, is an up and rising media production team that specializes in photography, videography and other creative services. They have been featured in City Gala 2016 at the Playboy Mansion, among many other events.

Overview

As the sole developer, I was tasked to enhance their online presence by providing responsive designs, build the front-end using jQuery, HTML, CSS and their back-end with PHP.

Mobile-Friendly

Foreground Studios provided me with desktop layouts for each page and relied on my experience to produce suitable layouts and solutions for mobile and tablet devices. I believe that it is important to provide an excellent experience to mobile and tablet devices, while making appropriate decisions to work around its limitations. I used media queries to serve a stylesheet specifically for mobile and tablet devices, such as using tabbed content to make it more mobile-friendly.

Filterable Content


An example of IsotopeJS

To allow the user to sort through content for each portfolio section, I used IsotopeJS in combination with MasonryJS to create a filterable layout for film categories, photo categories and design categories. To make sorting possible, each post is sorted by categories; when the user creates a post, they can create and choose multiple categories for the post to be categorized in. Within the front-end, the categories are listed as classes within each wrapper; when the user clicks on a category, I hide every wrapper that does not have the selected category within its list of classes.

Theme Settings


A look into Foreground's theme settings

Built into this WordPress theme are the theme settings, which allows the user to easily edit any features, content and styles on the website without the help of a webmaster. A few examples of what the user could edit would be social media links, the logo and any other related imagery, and the introductory text on the home page. Providing a theme settings page with the theme makes it easy for any user to edit the website, instead of having them directly edit the CSS and PHP files.

Conclusion

With my help and experience, we produced a smooth experience for users regardless of their preferred viewing device, which was what we planned to achieve during the design phase. I look forward to working more with Foreground Studios as the website and company grows.
