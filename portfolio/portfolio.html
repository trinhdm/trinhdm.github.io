Page Transitions

If you haven't noticed, when you click on a link on this website, the body fades out before jumping to the next web page. I never liked how jerky web pages jump from one to another (understandably, ReactJS can solve this problem), so I sought for a way to fix this problem. 

    $.fn.leavePage = function() {
      this.click(function(event){
        event.preventDefault();
        linkLocation = this.href;
        $('body').fadeOut(500, function(){
          window.location = linkLocation;
        });
      });
    };

    $('.transition').leavePage();

When a user clicks on a link, I call on a function that fades out the body before they go to the next page. When the user arrives on the next page, the body content fades in.
